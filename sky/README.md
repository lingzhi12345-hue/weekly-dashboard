# 光遇内容推广周报看板

Streamlit 可视化看板，支持数据上传和自动更新。

## 功能

- 📊 大盘数据可视化（抖音、小红书、B站、小喇叭）
- 📤 数据文件上传自动更新
- 📥 导出HTML报告
- 🔒 无数据板块自动折叠

## 本地运行

```bash
cd sky
pip install -r requirements.txt
streamlit run app.py
```

## 数据文件说明

将以下文件放入 `data/` 目录：

| 文件名 | 说明 |
|--------|------|
| 抖音-内容大盘-视频-作者分析-核心数据-*.csv | 抖音视频投稿UV、播放数据 |
| 抖音-内容大盘-直播-主播分析-核心数据-*.csv | 抖音直播开播、看播数据 |
| 抖音-内容大盘-视频-视频分析-视频榜单-*.csv | 抖音爆款视频榜单 |
| b站-花火品牌趋势-*.csv | B站品牌综合指数 |
| b站花火-内容趋势-*.csv | B站内容指数、播放指数等 |
| b站花火内容关联稿件-*.csv | B站爆款稿件 |
| 小红书-千瓜sky光遇品牌_每日数据.xls | 小红书笔记数、互动数据 |
| 小红书千瓜top10爆款.xls | 小红书爆款笔记 |
| 第一周小喇叭业务-大盘看板-*.xlsx | 小喇叭第一周数据 |
| 第二周小喇叭业务-大盘看板-*.xlsx | 小喇叭第二周数据 |

## 部署到 Streamlit Cloud

1. Fork 本仓库
2. 登录 [Streamlit Cloud](https://streamlit.io/cloud)
3. 创建新应用，选择 `sky/app.py`
4. 部署完成

## 更新数据

1. 在看板侧边栏点击"数据上传"
2. 上传新的数据文件
3. 文件会自动保存到 `data/` 目录并刷新看板
